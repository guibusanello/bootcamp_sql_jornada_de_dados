## Aula 2b - Operadores

Aula sobre os operadores do SQL.

### Operadores Aritméticos

- Adição: +
- Subtração: -
- Multiplicação: *
- Divisão: /
- Módulo: %

### Operadores de Comparação

- Igualdade: =
- Desigualdade: !=
- Maior que: >
- Menor que: <
- Maior ou igual a: >=
- Menor ou igual a: <=

### Operadores Lógicos

- AND
- OR
- NOT
- BETWEEN
- IN
- LIKE
- IS NULL
- IS NOT NULL
